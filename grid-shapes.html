(img:nth-child(2)) with the CSS clip-path property: clip-path: polygon(0 0, 100% 0, 100% 100%, calc(50% + var(5px) / 4) 100%, 0 calc(50% - var(5px) / 4)) And the top-right corner of the third one: clip-path: polygon(0 0, calc(50% - var(5px) / 4) 0, 100% calc(50% + var(5px) / 4), 100% 100%, 0 100%);
